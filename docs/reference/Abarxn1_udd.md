# UDD approximation of continuous term insurance

Computes \\\bar{A}\_{x:\overline{n}\|}^{1} =
(i/\delta)A\_{x:\overline{n}\|}^{1}\\.

## Usage

``` r
Abarxn1_udd(Axn1, i)
```

## Arguments

- Axn1:

  Discrete term insurance APV.

- i:

  Effective annual interest rate.

## Value

Continuous term insurance APV under UDD.
