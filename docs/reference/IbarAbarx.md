# Fully continuous increasing whole life insurance

Computes \$\$(\bar{I}\bar{A})\_x = \int_0^\infty t\\ v^t\\ {}\_tp_x\\
\mu\_{x+t}\\ dt.\$\$

## Usage

``` r
IbarAbarx(x, i, model, ...)
```

## Arguments

- x:

  Age.

- i:

  Effective annual interest rate.

- model:

  Parametric survival model.

- ...:

  Additional model parameters.

## Value

Numeric vector.
