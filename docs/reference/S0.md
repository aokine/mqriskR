# Survival function for age-at-failure T0

Computes the survival distribution function \\S_0(t)=Pr(T_0\>t)\\.

## Usage

``` r
S0(t, model, ...)
```

## Arguments

- t:

  Numeric vector of times (\\t \ge 0\\).

- model:

  One of `"uniform"`, `"exponential"`, `"gompertz"`, `"makeham"`,
  `"weibull"`.

- ...:

  Model parameters:

  - uniform: `omega`

  - exponential: `lambda`

  - gompertz: `B`, `c`

  - makeham: `A`, `B`, `c`

  - weibull: `shape`, `scale`

## Value

Numeric vector of survival probabilities in \\\[0,1\]\\.

## Details

Supported models (Chapter 5): uniform (de Moivre), exponential,
Gompertz, Makeham, Weibull.
