# mqriskR package

Provides functions for actuarial risk modeling, including survival
models, life annuities, multiple-decrement models, and mortality
improvement projections. The package is designed to align with standard
actuarial notation and supports teaching, exam preparation, and
reproducible actuarial analysis. The methods are based on standard
actuarial references including Camilli, Duncan and London (2014,
ISBN:9781625423474) "Models for Quantifying Risk" and Dickson, Hardy and
Waters (2020, ISBN:9781108478083) "Actuarial Mathematics for Life
Contingent Risks".
