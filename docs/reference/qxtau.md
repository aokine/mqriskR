# Total probability of decrement \\q_x^{(\tau)}\\

Total probability of decrement \\q_x^{(\tau)}\\

## Usage

``` r
qxtau(qxj)
```

## Arguments

- qxj:

  Numeric vector of cause-specific decrement probabilities.

## Value

Numeric scalar.

## Examples

``` r
qxtau(c(0.011, 0.100))
#> [1] 0.111
```
