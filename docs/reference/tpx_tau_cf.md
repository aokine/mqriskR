# Total survival probability \\{}\_t p_x^{(\tau)}\\ under constant forces

Total survival probability \\{}\_t p_x^{(\tau)}\\ under constant forces

## Usage

``` r
tpx_tau_cf(mu, t)
```

## Arguments

- mu:

  Numeric vector of cause-specific forces.

- t:

  Time.

## Value

Numeric scalar/vector.

## Examples

``` r
tpx_tau_cf(c(0.10, 0.20), 5)
#> [1] 0.2231302
```
