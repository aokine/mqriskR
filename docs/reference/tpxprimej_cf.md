# Single-decrement survival probability \\{}\_t p_x^{\prime(j)}\\ under constant force

Single-decrement survival probability \\{}\_t p_x^{\prime(j)}\\ under
constant force

## Usage

``` r
tpxprimej_cf(mu, t)
```

## Arguments

- mu:

  Force of decrement for cause j.

- t:

  Time.

## Value

Numeric scalar/vector.

## Examples

``` r
tpxprimej_cf(0.10, 5)
#> [1] 0.6065307
```
