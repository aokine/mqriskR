# Cause-specific probability \\{}\_t q_x^{(j)}\\ under constant forces

Cause-specific probability \\{}\_t q_x^{(j)}\\ under constant forces

## Usage

``` r
tqxj_cf(mu, j, t)
```

## Arguments

- mu:

  Numeric vector of cause-specific forces.

- j:

  Cause index.

- t:

  Time.

## Value

Numeric scalar/vector.

## Examples

``` r
tqxj_cf(c(0.10, 0.20), j = 1, t = 5)
#> [1] 0.2589566
```
