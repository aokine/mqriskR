# Single-decrement failure probability \\{}\_t q_x^{\prime(j)}\\ under constant force

Single-decrement failure probability \\{}\_t q_x^{\prime(j)}\\ under
constant force

## Usage

``` r
tqxprimej_cf(mu, t)
```

## Arguments

- mu:

  Force of decrement for cause j.

- t:

  Time.

## Value

Numeric scalar/vector.

## Examples

``` r
tqxprimej_cf(0.10, 5)
#> [1] 0.3934693
```
